# Exploratory Testing Session – Login Page

## Session Information

**Tester:** Amanda Bramble
**Date:** March 11, 2026
**Session Duration:** 30 minutes

---

## Testing Goal

Explore the login functionality to identify usability issues, validation problems, or unexpected behavior.

---

## Areas Explored

* Login form validation
* Error messages
* Empty input fields
* Incorrect credentials
* Login button functionality

---

## Observations

* The login button responds quickly when clicked.
* Error messages appear when incorrect credentials are used.
* Empty fields trigger validation messages.

---

## Potential Issues

* Email field allows leading spaces.
* Password field does not indicate password requirements.

---

## Notes

Further testing could include testing login behavior on mobile devices and different browsers.
